# Bharat Money Manager

A personal finance minimal MVP built with React (Vite), Express (Vercel), and Supabase.

## Architecture

- **Frontend:** React + Vite (deployed on Netlify)
- **Backend:** Node.js + Express (deployed on Vercel as serverless functions)
- **Database:** Supabase (PostgreSQL)

## Getting Started

### Prerequisites

- Node.js installed
- Supabase project credentials

### Execution

1. `cd frontend` -> `npm install` -> `npm run dev`
2. `cd backend` -> `npm install` -> `npm run dev` (run via nodemon or node index.js)
